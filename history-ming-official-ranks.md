# 🏛️ 明朝九品十八级官职图谱

> 九品十八级是中国古代官僚体系的核心制度，始创于魏晋时期，在明朝达到完备。九品分正、从二等，共十八个等级，正一品最高，从九品最低。

## 一、正一品 ～ 从二品（极品高层）

### 1.1 文官极品

| 品级 | 官职 | 职能 | 备注 |
|------|------|------|------|
| **正一品** | 太傅、太师、太保、光禄大夫（文） | 皇帝师傅，最高荣誉虚衔 | 极少授予，多为死后追赠 |
| **从一品** | 太子太傅、太子太师、柱国（勋） | 太子东宫官属、功勋最高级 | 武官同级的有"镇国公"等 |
| **正二品** | 太子少傅、太子少师、资政大夫、南京守备 | 太子东宫辅佐、中央要害、地方大员 | 掌中枢机要 |
| **从二品** | 布政使、都御史、祭酒、太常寺卿 | 省级最高行政长官（布政使） | 一省之治，左右布政使 |

**重点说明**：
- **太傅/太师/太保**：古称"三公"，明代多为**赠官**（死后追赠）或**荣衔**，实权在皇帝特旨
- **布政使**：省级最高行政长官，掌一省民政、财政，俗称"藩台"，与"按察使"（司法）、"都指挥使"（军事）并称**三司**

### 1.2 武官极品

| 品级 | 官职 | 职能 |
|------|------|------|
| **正一品** | 镇国公、辅国公（宗室）| 封爵，非战功不授 |
| **从一品** | 骠骑将军、金吾将军 | 禁军统领 |
| **正二品** | 副总兵、奉国将军（宗室）| 镇守一方 |
| **从二品** | 参将、昭勇将军 | 镇守副总兵 |

## 二、正三品 ～ 从四品（高级官员）

### 2.1 中央高官（正三品～正四品）

| 品级 | 官职 | 衙门 | 职能 |
|------|------|------|------|
| **正三品** | 都察院左右都御史、大理寺卿、鸿胪寺卿 | 都察院/大理寺 | 监察百官、审理大案 |
| **从三品** | 佥都御史、通政使司通政使 | 都察院/通政司 | 监察官员、章奏传达 |
| **正四品** | 金都御史、大理寺少卿、太常寺少卿 | 都察院/大理寺 | 副官 |
| **从四品** | 祭酒司业（国子监）、翰林院侍读学士 | 国子监/翰林院 | 教育、皇帝讲学 |

**核心衙门解析**：
```
┌─────────────────────────────────────────────┐
│              中央监察体系                        │
│                                             │
│  都察院（御史台）                            │
│    ├── 左右都御史（正三品）— 首长            │
│    ├── 左右副都御史（正四品）— 副首长       │
│    ├── 左右佥都御史（正五品）— 高级御史     │
│    └── 监察御史（正七品）— 基层监察        │
│                                             │
│  大理寺（最高法院）                          │
│    ├── 大理寺卿（正三品）— 首长            │
│    └── 大理寺少卿（正四品）— 副首长        │
│                                             │
│  刑部（公安部）                             │
│    └── 尚书（正二品）— 首长                │
└─────────────────────────────────────────────┘
```

### 2.2 地方大员（正三品～从四品）

| 品级 | 官职 | 辖区/职能 |
|------|------|-----------|
| **正三品** | 巡抚、总督（挂衔）| 巡视地方，节制镇守 |
| **从三品** | 布政使司左参议、按察使司副使 | 省级佐官 |
| **正四品** | 布政使司参政、按察使、各巡抚 | 省级要员 |
| **从四品** | 各府知府（标配）、运使 | 府级最高长官 |

**地方行政层级**：
```
省（布政使 / 从二品）
  └── 府（知府 / 从四品）
        └── 州（知州 / 从五品）
              └── 县（知县 / 正七品）
```

## 三、正五品 ～ 从六品（中级官员）

### 3.1 中央各部院（中层骨干）

| 品级 | 官职 | 所在衙门 | 职能 |
|------|------|---------|------|
| **正五品** | 郎中、员外郎、修撰 | 各部/翰林院 | 各部司级长官（郎中）或副官（员外郎）|
| **从五品** | 侍读、侍讲学士、洗马 | 翰林院/东宫 | 太子讲学侍读 |
| **正六品** | 主事、司务、大理寺评事 | 各部/大理寺 | 各部司级副官、基层审判官 |
| **从六品** | 行人司行人、都察院都事 | 行人司/都察院 | 捧诏书出使、章奏管理 |

**明代六部（吏户礼兵刑工）司级结构**：
```
┌─────────────────────────────────────────────┐
│              各部（尚书正二品/侍郎正三品）      │
│                                             │
│  某清吏司（某部某司）                        │
│    ├── 郎中（正五品）— 司长，实权官          │
│    ├── 员外郎（从五品）— 副司长              │
│    └── 主事（正六品）— 基层办事员            │
│                                             │
│  例：吏部文选清吏司                          │
│      → 郎中 1 人（正五品）                  │
│      → 员外郎 1 人（从五品）                │
│      → 主事 2 人（正六品）                  │
└─────────────────────────────────────────────┘
```

### 3.2 地方中层

| 品级 | 官职 | 所在 | 职能 |
|------|------|------|------|
| **正六品** | 府同知、直隶州知州 | 府/直隶州 | 知府佐官（分掌粮马/巡捕）|
| **从六品** | 通判、运副、翰林院检讨 | 府/运司/翰林院 | 佐官或史官 |

## 四、正七品 ～ 从八品（基层官员）

### 4.1 县级官员（地方基层核心）

| 品级 | 官职 | 职能 | 备注 |
|------|------|------|------|
| **正七品** | 知县、县级首官 | 一县之治，掌民政/财政/司法/科举 | 明代最基层的"亲民官" |
| **从七品** | 县丞、县级佐官 | 掌县丞分掌马政/水利 | 副县长 |
| **正八品** | 县尉、典史 | 掌治安/刑罚/监仓 | 武职县尉 / 文职典史 |
| **从八品** | 巡检司巡检 | 乡镇治安 | 驻守关津要道 |
| **正九品** | 县司主簿 | 掌文书/仓库 | 文秘总务 |
| **从九品** | 驿丞、税课司大使 | 驿站/税收 | 最基层杂职 |

**县级衙门结构**：
```
┌─────────────────────────────────────────────┐
│         知县（正七品）— 县令，亲民官          │
│                                             │
│  ├── 县丞（从七品）— 副官，分掌马政          │
│  ├── 主簿（正九品）— 文秘，掌符印、仓库      │
│  ├── 典史（从九品）— 掌监察/刑狱            │
│  ├── 县尉（正八品）— 掌治安（明代多虚设）  │
│  └── 巡检司（从八品）— 乡镇治安             │
└─────────────────────────────────────────────┘
```

## 五、特殊官职体系

```
│                                              │

│    ├── 巡按御史 — 代天子巡按地方，权力极重     │

│    ├── 巡茶御史 — 巡视茶马贸易                 │
│    └── 巡仓御史 — 巡视仓场                     │

│  特点：品低权重，以小制大，明太祖设计精妙        │

```

### 5.2 锦衣卫与特务机构

| 官职 | 品级 | 职能 |
|------|------|------|

| 锦衣卫指挥同知 | 从三品 | 副职 |

## 六、选官制度

### 6.1 科举正途

```
┌─────────────────────────────────────────────┐
│              科举取士流程                        │
├─────────────────────────────────────────────┤
│                                              │
│  童试（府/县试）──→  生员（秀才）            │
│      ↓                                        │
│  乡试（省试，每三年）──→  举人                │
│      ↓                                        │
│  会试（京师，乡试次年）──→  贡士              │
│      ↓                                        │
│  殿试（皇帝亲试）──→  进士（一甲/二甲/三甲） │
│                                              │
│  一甲3人：状元（正六品）/ 榜眼 / 探花         │
│  二甲若干：传胪（从七品）/ 其余（正七品）    │
│  三甲若干：同进士（从七品）                    │
└─────────────────────────────────────────────┘
```

### 6.2 荫补与捐纳

| 途径 | 说明 | 品级起点 |
|------|------|---------|
| **荫补** | 官员子弟不经科举，直接受荫补官 | 高官子弟可荫正五品 |
| **捐纳** | 富民纳粟买官（明中后期常见）| 虚衔或杂职 |
| **军功** | 凭军功授官，不拘品级 | 从九品至正三品不等 |

## 七、俸禄制度

### 7.1 俸禄等级（月俸，以米计）

| 品级 | 月俸（石米）| 约合银两（万历年间）|
|------|------------|---------------------|
| 正一品 | 87石 | ~87两 |
| 从一品 | 72石 | ~72两 |
| 正二品 | 61石 | ~61两 |
| 从二品 | 48石 | ~48两 |
| 正三品 | 35石 | ~35两 |
| 从三品 | 26石 | ~26两 |
| 正四品 | 24石 | ~24两 |
| 正七品（知县）| 16石 | ~16两 |
| 从九品 | 3石 | ~3两 |

> ⚠️ **"俸薄"问题**：明代官俸极低，知县月俸仅16石（约16两银），不足以维持体面生活，故贪腐成风。

## 八、历史演变

```
┌─────────────────────────────────────────────┐
│              明代官制演变                         │
├─────────────────────────────────────────────┤
│                                              │
│  洪武建政（1368）                            │
│    → 废丞相，皇权极度膨胀                      │
│    → 设内阁（最初只是顾问机构）                │
│    → 官制品级沿用九品十八级                   │
│                                              │
│  永乐～宣德（1402-1435）                     │
│    → 内阁权重上升，"票拟"制度形成             │
│    → 太监开始干政                             │
│                                              │
│  正德～嘉靖（1506-1566）                     │
│    → 厂卫特务机构权力鼎盛                     │
│    → 严嵩权臣时期                             │
│                                              │
│  万历之后（1573+）                           │
│    → 党争激烈，官场腐败                       │
│    → 官制品级制度名存实亡                     │
│    → 捐纳泛滥，官职贬值                       │
└─────────────────────────────────────────────┘
```

## 九、关键总结

### 九品十八级速记表

| 品级 | 文官代表 | 武官代表 | 核心特点 |
|------|---------|---------|---------|
| 正一品 | 太傅/太师 | 镇国公 | 极品荣衔，多为赠官 |
| 从一品 | 太子太傅 | 骠骑将军 | 荣衔/禁军统领 |
| 正二品 | 尚书 | 副总兵 | 各部首长/镇守大员 |
| 从二品 | 布政使 | 参将 | 省级最高/镇守副总兵 |
| 正三品 | 都御史 | 都指挥使 | 监察首长/地方军政 |
| 正四品 | 大理寺少卿 | 游击将军 | 司法副官/参将 |
| 正五品 | 郎中 | 守备 | 各部司级长官/府级军政 |
| 正六品 | 主事 | 卫镇抚 | 各部司级副官/县级佐官 |
| 正七品 | **知县** | 百户 | **亲民之官**/基层军职 |
| 正九品 | 主簿 | 司务 | 最基层文职/杂职 |

### 明代官制特点

```
┌─────────────────────────────────────────────┐
│              明代官制的三大特点                   │
├─────────────────────────────────────────────┤
│                                              │
│  1️⃣ 以小制大                                 │
│     监察御史正七品，可弹劾正一品大员            │
│     → 制度设计精妙，但执行依赖皇权支撑         │
│                                              │
│  2️⃣ 官俸极薄                                 │
│     正二品尚书月俸61石，不及宋代1/3            │
│     → 导致贪腐制度化，形成"常例"灰色收入      │
│                                              │
│  3️⃣ 衔职分离                                 │
│     实际职务（知府）与荣衔（资政大夫）分开     │
│     → 便于皇帝灵活调配，不受品级限制           │
└─────────────────────────────────────────────┘
```
